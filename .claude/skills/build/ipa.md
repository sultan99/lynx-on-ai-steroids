# Build iOS IPA

## Status: Not yet implemented

iOS build support has not been set up for this project yet.

## What's needed

1. Create an `ios/` directory with an Xcode project
2. Add Lynx iOS SDK via CocoaPods or Swift Package Manager
3. Create a `ViewController` that embeds `LynxView`
4. Load `main.lynx.bundle` from the app bundle
5. Build with `xcodebuild`

## When the user requests this

Inform them that iOS build is not yet configured and offer to help set it up.
