# /autopilot $ARGUMENTS

Activate auto-confirm mode — all skills skip confirmation gates for the rest of this session.

## Usage

```
/autopilot
```

## Instructions

### Step 1: Activate

Remember that **AUTO_CONFIRM is active** for the remainder of this conversation. All skills that check for confirmation gates must skip them and proceed automatically.

Say: "Autopilot enabled — all confirmations will be skipped for this session."
